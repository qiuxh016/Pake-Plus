# 功能 C：跨平台剪贴板管理实验报告

## 目标与范围

功能 C 为 Pake 增加可选的系统级文本剪贴板历史，同时保持原项目默认行为不变。用户只有在打包时加入 `--clipboard` 后才启用功能；`--clipboard-max` 控制 500～5000 条的容量，默认 2000 条。历史记录会尽可能采集来源应用名称，面板内提供独立的隐私过滤设置；成员 D 后续仍可通过相同设置 IPC 将这些选项整合到统一设置窗口。

## 模块架构

实现位于 `src-tauri/src/app/clipboard/`。`monitor` 负责跨平台监听和写回竞态，`filter` 负责隐私规则，`store` 管理 SQLite，`settings` 管理运行时配置，`commands` 提供受权限控制的 IPC，`panel` 管理独立面板窗口。CLI 将参数写入 `pake.json`，Rust 启动时以打包参数覆盖功能启用状态和容量上限，同时保留本地隐私过滤设置。功能关闭时不会编译剪贴板依赖，也不打开监听器或新建数据库。

## 监听、去重与生命周期

Windows、macOS 和 X11 使用 `clipboard-rs 0.3.5` 的系统变化事件，不持续读取剪贴板。Windows 通过剪贴板所有者窗口解析进程名，macOS 读取前台应用，Linux/X11 通过 `_NET_ACTIVE_WINDOW` 与 `_NET_WM_PID` 解析进程名。Linux 检测到仅有 `WAYLAND_DISPLAY`、没有 `DISPLAY` 时，因协议本身既不提供全局变化通知，也不公开其他客户端的来源应用，自动降级为 300ms 文本轮询并将来源留空。轮询启动时只保存当前文本哈希，不把启动前内容写入历史。面板复用记录时先登记“预期哈希”，监听回调收到相同哈希便消费并跳过，避免复制动作与系统事件竞争。监听器可停止、重建和按配置更新，应用退出时会发送停止信号并回收线程。

## 数据库、搜索与隐私

历史库路径为应用配置目录下的 `clipboard/history.db`。每条数据保存正文、SHA-256 哈希和 RFC 3339 时间；哈希具有唯一索引，重复复制会更新时间并回到列表顶部。启动、新增记录和独立的每小时定时任务都会清理 30 天前数据。达到上限后按固定 500 条批次删除最旧记录。

入库前默认拒绝：少于 2 字符、超过 10000 字符、6～30 位纯字母数字，以及符合银行卡分组格式的文本。用户可以在面板设置中分别关闭短文本、密码样式和银行卡样式过滤，也可以逐行填写不记录的来源应用名称；应用匹配忽略大小写及 `.exe`/`.app` 后缀。搜索将空格分隔的词按 AND 组合，使用小写包含匹配，中文可直接检索；反斜杠、`%`、`_` 会转义，避免被 SQLite 当成通配符。

## 面板、快捷键与安全

面板为 320×480、无边框、置顶的右下角窗口，可按系统语言显示中英文。记录显示两行预览、相对时间和检测到的来源应用；悬停时显示“复制”“展开”“删除”按钮，单击记录本身仍可立即复制并关闭。面板支持即时搜索、隐私过滤设置、统计、二次确认清空，并在 ESC、失焦或关闭请求时隐藏。快捷键为 Windows/Linux 的 `Ctrl+Shift+V` 和 macOS 的 `Cmd+Shift+V`；若与应用激活快捷键冲突，剪贴板入口优先并输出警告。启用托盘时显示监听圆点、状态提示和备用菜单入口。

面板 HTML 通过 `pake-clipboard` 自定义协议内嵌返回，并校验请求 webview 标签，不再使用 `about:blank + document.write`。`clipboard-panel` 能力只授权历史与剪贴板隐私设置命令；普通 `pake` 远程网页能力不含任何剪贴板权限。设置命令仍使用独立权限标识，成员 D 后续可以把该权限授予受信任的统一设置 webview。

## 测试结果与问题处理

单元测试覆盖过滤边界、可开关的密码与银行卡规则、来源应用忽略匹配、来源字段持久化、面板复制与设置入口、去重、中文/英文 AND 搜索、通配符转义、删除清空、30 天清理、500 条容量批量清理和预期哈希竞态。Windows 本机负责 Rust 与面板测试；macOS、Linux 仍需由三平台 CI 和对应系统人工验证来源检测及纯 Wayland 降级分支。

## Windows 手工验收与截图清单

验收依次覆盖：外部应用复制后 500ms 内出现；短文本、疑似密码和银行卡被跳过；快捷键与托盘均能打开面板；中英文搜索、展开、删除、清空正常；点击记录后可在外部应用粘贴；重启后历史仍在；按 ESC 或右上角关闭面板；禁用后不再新增记录。报告截图建议包含：CLI 打包参数、托盘监听状态、历史列表、中文搜索、长文本展开、清空确认、敏感内容未入库、重启持久化和数据库路径。
